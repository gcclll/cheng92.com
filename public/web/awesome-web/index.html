  1. 此文就是为了方便查看库的星级和 build status 而产生。

  2. 文中表格的实现基于 vue-next + element-plus

  3. 鼠标放在英文简介上面会出现中文翻译（自己+google翻译的仅供参考）。

  4. 鼠标放到表中名称链接上面会展示预览图或GIF动画(部分有)

  5. 描述前面如果有一个链接说明这个库有自己的文档网站
